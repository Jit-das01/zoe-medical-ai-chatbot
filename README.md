# ZOE 2.0 — AI Medical Chatbot

An AI-powered medical chatbot built with Google Gemini API.
Supports voice input/output, PWA, and ChatGPT-style UI.

![Status](https://img.shields.io/badge/Status-Live-brightgreen)
![Python](https://img.shields.io/badge/Python-3.10+-blue)
![React](https://img.shields.io/badge/React-18-blue)
![AI](https://img.shields.io/badge/AI-Gemini_API-orange)

## Features
- Voice input and output (hands-free)
- Google Gemini API powered responses
- Progressive Web App (PWA) — installable
- ChatGPT-style conversational UI
- Medical Q&A focused

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React, Web Speech API |
| Backend | Python, FastAPI |
| AI | Google Gemini API |
| Deployment | Replit |

## Status
Currently available via GitHub. Mobile version built with Expo/React Native.
Clone the repo and run locally using the setup instructions below.

## Setup
```bash
git clone https://github.com/Jit-das01/zoe-medical-ai-chatbot
cd zoe-medical-ai-chatbot
bash setup.sh
```
